# RiProG Launcher (v3.0.1)

RiProG Launcher is an ultra-lightweight, high-performance Android launcher designed for simplicity, speed, and a distraction-free user experience. It prioritizes efficiency and a modern aesthetic, providing a smooth interface even on lower-end devices.

---

## 🚀 Key Features

### ⚡ Exceptional Performance
*   **Minimal Footprint:** Optimized APK size and extremely low RAM utilization.
*   **Fast Execution:** Engineered for rapid app launching and smooth navigation.
*   **Programmatic UI:** The entire interface is built programmatically to eliminate XML inflation overhead, ensuring maximum responsiveness.

### 🧊 Acrylic UI Aesthetic
*   **Consistent Design:** Features a sophisticated acrylic UI across the app drawer, settings, and system menus.
*   **Visual Harmony:** Transparent and translucent elements that adapt to your wallpaper for a modern look and feel.

### 🔍 Advanced App Drawer
*   **Intelligent Search:** Quickly locate applications with a built-in search bar.
*   **Alphabetical Quick Index:** An efficient scroll bar for rapid navigation through large lists of applications.
*   **Swipe-to-Open:** Seamlessly transition from your home screen to your apps with a simple swipe gesture.

### 🛠️ Customization & Flexibility
*   **Advanced Widget UI:** A dedicated widget selection interface with visual shape previews for easy identification and placement.
*   **Precision Freeform Layout:** An optional mode that allows you to bypass traditional grid constraints, giving you the freedom to place apps, folders, and widgets anywhere on your home screen.
*   **Dynamic Scaling:** Real-time adjustment of icon and label sizes to perfectly suit your visual preferences.
*   **Folder Support:** Organize your home screen by grouping related apps into custom folders.

### 🎨 Intelligent Theming
*   **Adaptive Modes:** Full support for Light, Dark, and System-following themes.
*   **Seamless Transitions:** UI elements transition smoothly between theme modes.
*   **Wallpaper Dimming:** Optional setting to darken your wallpaper in dark mode for better readability and reduced eye strain.

---

## 📱 Application Structure & Behavior

RiProG Launcher is structured to be intuitive and lightweight. The main home screen serves as your primary workspace, supporting widgets and app shortcuts.

*   **Home Screen:** Supports multiple pages, freeform placement (optional), and easy management of shortcuts.
*   **App Drawer:** Accessed via a swipe up, it organizes all installed apps alphabetically with a powerful search function at the top.
*   **Launcher Settings:** Accessible via a long-press on the home screen, allowing you to customize theme modes, styles (Standard vs. Acrylic), and home screen behavior.
*   **Emergency Tools:** Includes options for force-restarting the launcher and erasing all data/settings to revert to a clean state if needed.

---

## 🛠 Technical Specifications

*   **Minimum SDK:** API 23 (Android 6.0)
*   **Target SDK:** API 36 (Android 16)
*   **Implementation:** Fully Kotlin-based, utilizing programmatic UI construction for performance.
