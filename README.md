# RiProG Launcher v2.0.1

Ultra-lightweight Android launcher. Minimal, fast, and distraction-free.

## Features
- **Ultra-Lightweight**: Minimal APK size and RAM footprint.
- **Fast & Responsive**: Optimized for speed and low CPU usage.
- **App Drawer with Search**: Quickly find and launch your apps.
- **Gesture Support**: Simple swipe up/down to access the app drawer.
- **Widget Support**: Add your favorite widgets to the home screen.
- **Theme Support**: Dark/Light mode compatibility.
- **Privacy Focused**: No unnecessary permissions or tracking.

## Technical Details
- **Min SDK**: 23 (Android 6.0)
- **Target SDK**: 34 (Android 14)
- **Language**: Java 17
- **Architecture**: Lightweight programmatic UI (No XML inflation).
