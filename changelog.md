Changelog
- Added support for home screen folders to group applications.
- Introduced a modern Acrylic UI style with real-time blur and reflection effects.
- Implemented a dedicated Widget Picker activity for easier widget selection.
- Added wallpaper dimming support for dark mode.
- Added home screen page management with "Add Left" and "Add Right" options.
- Replaced broad QUERY_ALL_PACKAGES permission with specific queries for improved privacy.
- Improved automatic cleanup of orphaned home screen items from uninstalled apps.
- Added triple-tap confirmation to prevent accidental data erasure or force restarts.
